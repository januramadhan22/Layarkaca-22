# Layarkaca 22

<strong>This project is an online Movies Galleries .<strong>

</div>

## Feature

- Home <br>
  Users can see list of all already movies, and user can add every movies to favorite.

- Favorites Page <br>
  Users can see list of all favorite movies has been added.

- Detail Movies Page <br>
  users can see the details of the film, such as synopsis, release date, duration of the film and others.
